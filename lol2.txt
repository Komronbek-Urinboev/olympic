

## Задача

В стране имеется **K государств** и в сумме **N городов**.

* В первом государстве расположено `a1` городов, во втором — `a2` городов, …, в K-м — `aK` городов.
* Города внутри каждого государства перенумерованы подряд:

  * города 1-го государства: с 1 по `a1`;
  * города 2-го государства: с `a1+1` по `a1+a2`;
  * города K-го государства: с `a1+...+a_{K-1}+1` до `N`.

Между соседними городами `(i, i+1)` внутри одного государства всегда существует дорога.

Кроме того, даны **M дополнительных дорог**, каждая из которых соединяет два города из разных государств.

---

### Запросы

Вам задаются **Q запросов**. В каждом запросе даны два города **X** и **Y**, которые гарантированно находятся в разных государствах.

Нужно найти **количество кратчайших путей** между городами **X** и **Y**.

Так как ответ может быть очень большим, выведите его по модулю `10^9 + 7`.

---

### Входные данные

1. В первой строке даны три числа:

   ```
   K N M   (1 ≤ K ≤ N ≤ 3*10^5, 1 ≤ M ≤ 3*10^5)
   ```

2. Во второй строке даны числа:

   ```
   a1, a2, …, aK   (a1 + a2 + ... + aK = N)
   ```

3. Следующие **M** строк содержат по два числа `u v (1 ≤ u, v ≤ N, u ≠ v)`, описывающих дополнительную дорогу.
   Гарантируется, что каждая такая дорога соединяет города из разных государств.

4. Следующая строка содержит число **Q** `(1 ≤ Q ≤ 3*10^5)` — количество запросов.

5. В следующих **Q** строках заданы пары `(X Y)` `(1 ≤ X, Y ≤ N, X и Y в разных государствах)`.

---

### Выходные данные

Для каждого запроса выведите одно число — количество кратчайших путей из `X` в `Y` по модулю `10^9+7`.

---

### Пример

**Входные данные:**

```
3 13 5
4 5 4
1 3
2 4
5 7
6 8
10 13
3
7 13
1 4
1 13
```

**Выходные данные:**

```
2
1
4
```
